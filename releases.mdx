## Checking Your Version

To check which version of ACE Copilot you're currently running:

<Steps>
  <Step title="Open ACE Copilot">
    Launch the application from your desktop or Start menu
  </Step>
  <Step title="Access Help Menu">
    Click on the **Help** menu in the top navigation bar
  </Step>
  <Step title="Select About">
    Choose **About ACE Copilot** to view your current version number
  </Step>
</Steps>

The version number will be displayed prominently in the About dialog.

## Update Notifications

ACE Copilot automatically checks for updates when you launch the application. When a new version is available:

- A notification banner appears at the top of the main window
- You can choose to update immediately or postpone
- Updates are downloaded directly from GitHub releases
- Your existing data and settings are preserved during updates

<Tip>
  Enable automatic update checks in **Settings > General > Check for updates on startup** to stay current with the latest features and security improvements.
</Tip>

## Upgrade Instructions

<Steps>
  <Step title="Download Latest Version">
    Download the latest ACE Copilot installer from the link above
  </Step>
  <Step title="Close Current Application">
    Close any running instances of ACE Copilot
  </Step>
  <Step title="Run Installer">
    Double-click the downloaded installer and follow the prompts
  </Step>
  <Step title="Automatic Migration">
    The installer will automatically detect and migrate your existing data
  </Step>
  <Step title="Verify Installation">
    Launch ACE Copilot and verify your version number in Help > About
  </Step>
</Steps>

<Warning>
  Always back up your data before upgrading. Use **File > Export Project** to create a backup of your current work.
</Warning>
