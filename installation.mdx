# Installing ACE Copilot

This guide will walk you through downloading and installing ACE Copilot version 2.3 on your Windows system.

## System Requirements

Before installing, verify that your system meets the minimum requirements:

<AccordionGroup>
  <Accordion title="Minimum Requirements">
    - **Operating System**: Windows 10 (64-bit) or later
    - **Processor**: Intel Core i3 or AMD equivalent
    - **Memory**: 4 GB RAM
    - **Storage**: 100 MB available disk space
    - **Display**: 1280x720 resolution
  </Accordion>
  
  <Accordion title="Recommended Requirements">
    - **Operating System**: Windows 11 (64-bit)
    - **Processor**: Intel Core i5 or AMD equivalent
    - **Memory**: 8 GB RAM or more
    - **Storage**: 500 MB available disk space
    - **Display**: 1920x1080 resolution or higher
  </Accordion>
</AccordionGroup>

<Warning>
  ACE Copilot requires a 64-bit version of Windows. 32-bit systems are not supported.
</Warning>

## Download ACE Copilot

ACE Copilot is distributed as a standalone executable file (~46MB) available through GitHub releases.

<Steps>
  <Step title="Navigate to GitHub Releases">
    Visit the official ACE Copilot releases page:
    
    ```
    https://github.com/alfonsovelardem/ace-copilot/releases
    ```
    
    <Note>
      Always download from the official GitHub repository to ensure you're getting the authentic, unmodified application.
    </Note>
  </Step>
  
  <Step title="Download the Installer">
    Download the latest version (v2.3) executable file:
    
    [Download AceCopilot.exe](https://github.com/alfonsovelardem/ace-copilot/releases/download/AceCopilotV1.2/AceCopilot.exe)
    
    The file size is approximately **46 MB**. Download time will vary based on your internet connection speed.
  </Step>
  
  <Step title="Verify the Download">
    Once downloaded, verify that:
    - File name: `AceCopilot.exe`
    - File size: ~46 MB
    - File location: Your Downloads folder (or chosen location)
  </Step>
</Steps>

## Installation Process

<Steps>
  <Step title="Locate the Downloaded File">
    Navigate to your Downloads folder (or wherever you saved the file) and locate `AceCopilot.exe`.
  </Step>
  
  <Step title="Run the Installer">
    Double-click on `AceCopilot.exe` to launch the application.
    
    <Warning>
      Windows may display a security warning since this is a downloaded executable. Click "More info" and then "Run anyway" to proceed with the installation.
    </Warning>
  </Step>
  
  <Step title="Grant Administrator Permissions">
    If prompted by User Account Control (UAC), click **Yes** to allow the application to make changes to your device.
    
    <Note>
      Administrator permissions may be required for initial setup and certain features.
    </Note>
  </Step>
  
  <Step title="Complete First-Time Setup">
    Follow any on-screen prompts to complete the initial configuration. This may include:
    - Accepting terms and conditions
    - Choosing installation preferences
    - Configuring initial settings
  </Step>
</Steps>

## Post-Installation Verification

After installation, verify that ACE Copilot is working correctly:

<Steps>

  </Step>
  
  <Step title="Check Version Information">
    Verify you're running version 2.3 by checking the application's About or Help menu.
  </Step>

## Common Installation Issues

<AccordionGroup>
  <Accordion title="Windows SmartScreen Warning">
    **Issue**: Windows displays "Windows protected your PC" message.
    
    **Solution**: Click "More info" and then "Run anyway". This warning appears for applications that aren't widely recognized by Microsoft SmartScreen.
  </Accordion>
  
  <Accordion title="Application Won't Launch">
    **Issue**: Double-clicking the executable doesn't launch the application.
    
    **Solution**: 
    - Ensure you have administrator privileges
    - Check that your Windows version is up to date
    - Try running as administrator (right-click > Run as administrator)
    - Verify your system meets the minimum requirements
  </Accordion>
  
  <Accordion title="Missing Dependencies">
    **Issue**: Error messages about missing DLL files or components.
    
    **Solution**:
    - Install the latest Windows updates
    - Install Microsoft Visual C++ Redistributable packages
    - Restart your computer and try again
  </Accordion>
  
  <Accordion title="Antivirus Blocking Installation">
    **Issue**: Your antivirus software blocks or quarantines the executable.
    
    **Solution**:
    - Add an exception for AceCopilot.exe in your antivirus settings
    - Temporarily disable antivirus during installation (re-enable after)
    - Download from the official GitHub repository only
  </Accordion>
</AccordionGroup>

## Updating ACE Copilot

To update to a newer version:

1. Download the latest release from GitHub
2. Close any running instances of ACE Copilot
3. Run the new installer
4. Your settings and data should be preserved

<Note>
  Always check the release notes for breaking changes or migration steps before updating.
</Note>

## Uninstallation

If you need to uninstall ACE Copilot:

1. Close all running instances of the application
2. Open Windows Settings > Apps > Installed apps
3. Search for "ACE Copilot"
4. Click the three dots menu and select "Uninstall"
5. Follow the prompts to complete uninstallation

Alternatively, you can simply delete the executable file if no formal installation was performed.
